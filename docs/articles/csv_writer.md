# CsvWriter

**Description**  
The `CsvWriter` node handles writing frame metadata and associated numerical data to `.csv` files.

**Properties**

- **FileName**  
Specifies the full file path and name of the output `.csv` file.

- **Suffix**  
Specifies the suffix appended to generated file names.

- **Overwrite**  
Indicates whether the output file should be overwritten if it already exists.
